# AdventOfCode2022

https://adventofcode.com/

Repository for advent of code 2022.

### How to run

This project use ts-node to run typescript files.
Install it with `npm i ts-node`.

Then do `npm install` and run a file like this `ts-node .\day2\b.ts`
